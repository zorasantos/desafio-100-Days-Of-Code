## Regras Adicionais
1. Eu vou pubicar no story do Instagram(não uso twitter) sobre o meu progresso todos os dias -> usando a hashtag # 100DaysOfCode
2. Se eu codificar no trabalho, esse tempo não contará para o desafio.
3. Eu vou empurrar o código para o GitHub todos os dias para que qualquer um possa ver meu progresso.
4. Vou atualizar o log .[log.md].(https://github.com/zorasantos/desafio-100-Days-Of-Code/blob/master/Log.md) com o progresso do dia e fornecer um link para que outros possam ver o meu progresso.
5. Eu vou trabalhar em projetos reais, enfrentando desafios reais. O tempo gasto fazendo tutoriais, cursos on-line e outros recursos semelhantes NÃO serão contados para esse desafio.
